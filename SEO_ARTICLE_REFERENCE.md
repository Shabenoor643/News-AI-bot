# 📝 SEO Article Structure Reference

## Article Template (1200-2000 words)

### 1. SEO Metadata
```
Title: [60-70 chars, keyword-rich, high CTR]
Meta Description: [150-160 chars, compelling, includes primary keyword]
Slug: [clean-url-with-primary-keyword]
```

**Example**:
```
Title: Royal Enfield Himalayan 450 Review: Best Adventure Bike Under ₹3L?
Meta: Comprehensive Royal Enfield Himalayan 450 review covering specs, performance, price, and real-world testing. Is it India's best adventure bike?
Slug: royal-enfield-himalayan-450-review-2024
```

---

### 2. Introduction (150-200 words)

**Structure**:
- **Hook** (question or bold statement)
- **Context** (why this matters)
- **Promise** (what reader will learn)

**Example**:
```markdown
Is the Royal Enfield Himalayan 450 the adventure bike India has been waiting for? 

With adventure motorcycling gaining massive popularity in India, Royal Enfield's 
latest offering promises to redefine the segment with its all-new 450cc engine, 
modern electronics, and competitive pricing under ₹3 lakh.

In this comprehensive review, we'll dive deep into the Himalayan 450's performance, 
features, build quality, and real-world usability to help you decide if this is 
the right adventure companion for your journeys.
```

## SEO Checklist

### Keywords
- [ ] Primary keyword in title
- [ ] Primary keyword in first paragraph
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in conclusion
- [ ] Secondary keywords distributed naturally
- [ ] 3-5 long-tail variations included

### Structure
- [ ] Proper H1 → H2 → H3 hierarchy
- [ ] Short paragraphs (3-4 lines max)
- [ ] Bullet points for lists
- [ ] Tables for comparisons/specs
- [ ] Internal link placeholders
- [ ] FAQ section included

### Content Quality
- [ ] 1200+ words (strict minimum)
- [ ] No generic filler
- [ ] Data-driven (specs, numbers, dates)
- [ ] Indian context (₹ pricing)
- [ ] Engaging, conversational tone
- [ ] Clear explanations
- [ ] Real value (examples, insights)

### Readability
- [ ] Clear introduction with hook
- [ ] Table of contents
- [ ] Logical section flow
- [ ] Pros/cons clearly listed
- [ ] Strong conclusion with CTA

## Common Mistakes to Avoid

❌ **Don't**:
- Write short summaries (must be 1200+ words)
- Stuff keywords unnaturally
- Use generic filler content
- Skip the FAQ section
- Forget Indian context (₹ pricing)
- Hallucinate specifications
- Use complex jargon without explanation

✅ **Do**:
- Write comprehensive, valuable content
- Use keywords naturally
- Include real data and specs
- Add FAQ for SEO
- Focus on Indian market
- Verify all facts from sources
- Explain technical terms clearly

## Meta Description Formula

**Format**: `[Action] [Primary Keyword] [Benefit/Feature] [Call-to-action]`

**Examples**:
- "Discover the Royal Enfield Himalayan 450 specs, price, and performance. Complete review with real-world testing. Read now!"
- "Compare the best mileage bikes in India under ₹1.5 lakh. Detailed specs, pricing, and fuel efficiency data. Find your perfect bike!"
- "Bajaj Pulsar NS200 review covering performance, features, and value. Is it the best 200cc bike? Find out here!"

**Rules**:
- 150-160 characters
- Include primary keyword
- Compelling, actionable
- Clear benefit/promise
